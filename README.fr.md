**Codex Skill Forge** est un package public de skill et de plugin Codex pour créer des skills de haute qualité à partir de briefs, d'exemples, de fichiers et d'instructions concrètes.

## Ce Qu'il Produit

- des descriptions `SKILL.md` précises et faciles à déclencher
- des workflows courts avec divulgation progressive
- des scripts, références, assets et métadonnées de plugin quand ils améliorent la fiabilité
- validation, forward-testing et publication sûre
- chemins d'installation et dry-run pensés d'abord pour Windows

## Démarrage Rapide

```powershell
git clone https://github.com/ucsahinn/codex-skill-forge.git
cd codex-skill-forge
npm run validate
```
