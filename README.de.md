**Codex Skill Forge** ist ein öffentlich nutzbares Codex-Skill- und Plugin-Paket zum Erstellen hochwertiger Codex-Skills aus Briefings, Beispielen, Dateien und klaren Vorgaben.

## Was Es Erstellt

- präzise und gut triggerbare `SKILL.md`-Beschreibungen
- schlanke Workflows mit Progressive Disclosure
- optionale Skripte, Referenzen, Assets und Plugin-Metadaten
- Validierung, Forward-Testing und sichere Veröffentlichung
- Windows-first Installations- und Dry-Run-Pfade

## Einstieg

```powershell
git clone https://github.com/ucsahinn/codex-skill-forge.git
cd codex-skill-forge
npm run validate
```
