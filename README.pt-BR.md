**Codex Skill Forge** é um pacote público de skill e plugin para Codex que ajuda a criar skills de alta qualidade a partir de briefs, exemplos, arquivos e instruções concretas.

## O Que Ele Cria

- descrições `SKILL.md` claras e fáceis de acionar
- workflows enxutos com progressive disclosure
- scripts, referências, assets e metadados de plugin quando eles aumentam a confiabilidade
- validação, forward-testing e publicação segura
- caminhos de instalação e dry-run pensados primeiro para Windows

## Começo Rápido

```powershell
git clone https://github.com/ucsahinn/codex-skill-forge.git
cd codex-skill-forge
npm run validate
```
