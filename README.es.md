**Codex Skill Forge** es un paquete público de skill y plugin para Codex que ayuda a crear skills de alta calidad a partir de briefs, ejemplos, archivos e instrucciones concretas.

## Qué Construye

- descripciones `SKILL.md` claras y fáciles de activar
- workflows compactos con divulgación progresiva
- scripts, referencias, assets y metadatos de plugin cuando aportan valor real
- validación, forward-testing y publicación segura
- rutas de instalación y dry-run pensadas para Windows

## Inicio Rápido

```powershell
git clone https://github.com/ucsahinn/codex-skill-forge.git
cd codex-skill-forge
npm run validate
```
