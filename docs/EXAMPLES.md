# Examples

This repository includes three small examples:

- [minimal-skill](../examples/minimal-skill/SKILL.md): instruction-only release notes helper.
- [scripted-skill](../examples/scripted-skill/SKILL.md): skill with a deterministic name-normalizer script.
- [plugin-packaged-skill](../examples/plugin-packaged-skill/plugin.json): small plugin manifest example.

Use these examples as shape references, not as universal templates.
